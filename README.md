# Network_Analysis
## IT University of Copenhagen, 2025

## Description

- Dataset Name: Bitcoin OTC Trust Weighted Signed Network

- Source: https://snap.stanford.edu/data/soc-sign-bitcoinotc.html

**Research Question**

_Do negative ratings occur more frequently across different trust-based communities
than within the same community?_

This question explores social polarization and trust fragmentation in peer-to-peer
financial systems.

 **Methods:**

- Data Preprocessing & Exploratory Data Analysis

- Community Detection

- Cross-Community Distrust Analysis

- Visualization
